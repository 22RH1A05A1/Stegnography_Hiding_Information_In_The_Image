Image Steganography - Hide and Retrieve Messages in Images

Usage:
1. To hide a message:
   > python encrypt.py
   Enter input image path: sample.png
   Enter output image path: encoded.png
   Enter the message to hide: Hello World

2. To retrieve the hidden message:
   > python decrypt.py
   Enter encoded image path: encoded.png

Dependencies:
- Python 3
- Pillow

Install requirements:
> pip install -r requirements.txt
